## 1. 프로젝트 소개

## 2. 개발 환경

* **서버** 
  * Apache-tomacat-8.5
  
* **Java EE IDE** 
  * Eclipse
  
* **DB** 
  * Oracle SQL Developer 11g
  
* **개발 언어** 
  * **Back-end** 
    * Java (jdk 1.8)
      *  JSP (JSTL) 
  * **Front-end** 
    * HTML
    * CSS
    * JavaScript
      * Jquery 3.5
       
  
* **프레임워크**
  * Spring Mybatis Legacy
  * maven
  
* **라이브러리**
  * Bootstrap 5
  * Spring Sockets (WebSocket, SockJS, Stomp)
  * Commons Email
  * CKeditor5
 
* **형상관리**
  * GithuB
  
* **외부 API**
  * Kakao Login REST API
  * ZOOM API
  * Duam 우편번호 찾기 API
  
* **Remote Hosting**
  * AWS EC2 Window

# 주요 기능

 1.	**게시판 페이지**
    * 글쓰기 / 글 수정 / 글 삭제 / 댓글 작성 / 댓글 수정 / 댓글 삭제/ (CRUD) / 좋아요 기능/ 검색 기능 / Paging 처리
 
 2.	**회원 페이지** 
    * 회원가입 / 회원 정보 수정 / 탈퇴 / 아이디 및 비밀번호 찾기
 
 3.	**채팅**  
    * 친구 추가 / 1:1 채팅 / 단체 채팅 / Zoom 화상 회의 초대
 
 4.	**화상 회의 게시판** 
    * Zoom API를 활용한 화상 회의
